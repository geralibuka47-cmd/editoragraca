# Import Data

Este workflow ajuda a sincronizar dados com o Firebase usando os scripts de importação.

// turbo

1. Execute o script de importação para o Firebase:

```powershell
node scripts/import-to-firebase.js
```

> [!NOTE]
> Certifique-se de que as variáveis de ambiente no `.env` estão configuradas corretamente para o ambiente de destino.
